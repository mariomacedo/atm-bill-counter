Cite as diferenças e responsabilidades entre: HTML, CSS, Javascript

HTML - HyperText Markup Language

É responsável por criar as páginas web. A estrutura da página é definida através de tags e os componentes são interpretados pelo navegador.

CSS - Cascading Style Sheet

Primeiramente responsável por formatar e estilizar os elementos HTML. Porém, atualmente evoluiu para uma pseudo linguagem de programação. Através do css dá-se movimento, cor, espaçamento e etc, aos elementos da página.

Javascript

É responsável por dar dinâmica e ação às páginas e aplicações web. É uma linguagem de programação interpretada, baseada em orientação a objetos e client-side. Utilizada principalmente em navegadores web, mas também em outros ambientes como Node.js. É uma linguagem de script multiplataforma que evolui rápido e torna-se cada vez mais poderosa. F
