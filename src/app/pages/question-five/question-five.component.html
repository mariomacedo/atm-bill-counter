Explique a diferença entre uma arquitetura MVC e SPA (Ex: AngularJs)

MVC - Model View Controller

O padrão de projeto MVC divide a aplicação em três subsistemas

  • Model - Responsável pelo encapsulamento dos dados
  • View - Obtenção e apresentação dos dados. Geralmente a interface com o usuário.
  • Controller - Faz o intermédio e a modificação dos dados.

A regra de negócio fica isolada dos dados, que por sua vez ficam isolados da interface com usuário. Dividindo-se as responsabilidades e aumentando a manutenibilidade da aplicação.

SPA - Single Page Application

Ao contrário das MPA (Multi Page Application), as SPAs não fazem requisição de uma página inteira. SPA reescreve porções de uma única página, com o objetivo de passar a experiência de um aplicativo ao usuário.
